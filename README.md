# Embroidery Factory Website

A professional multi-page business website developed for an embroidery design and computerized embroidery manufacturing factory.
The website showcases embroidery services, design portfolio, company information, and contact details for potential clients.

## 🌐 Live Website

https://yashsb-07.github.io/embroidery-factory-website/

## 📌 Features

* Responsive multi-page website
* Modern homepage with hero image slider
* Embroidery design gallery with lightbox image preview
* Services and About pages
* Contact form section
* Sticky navigation bar with mobile hamburger menu
* Smooth scrolling and hover animations
* Professional branding (logo, color palette, typography)

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Git & GitHub
* GitHub Pages (Deployment)
